Image Classification using Convolutional Neural Network (Nike vs Adidas)
Project Overview

This project implements a Convolutional Neural Network (CNN) to classify images as either Nike or Adidas based on logo features. The model automatically extracts image features using convolutional layers and performs classification using deep learning techniques.

The objective of this project is to demonstrate an end-to-end deep learning workflow including image preprocessing, CNN model development, training, evaluation, and prediction.

Objective

To build a deep learning image classification model that can distinguish between Nike and Adidas logos using a Convolutional Neural Network architecture.

Dataset

The dataset consists of labeled images belonging to two categories:

Nike
Adidas

Each image is converted to grayscale and resized to a fixed dimension of:

120 × 120 pixels

This ensures uniform input size for training the CNN model.

Data Preprocessing Steps

Images converted to grayscale format
Images resized to 120 × 120 resolution
Labels encoded using one-hot encoding
Dataset shuffled to avoid training bias
Training data stored as NumPy arrays

Model Training

The model was trained using:

Optimizer: Adam
Loss Function: Categorical Crossentropy
Evaluation Metric: Accuracy

Training performed over multiple epochs to allow the model to learn hierarchical image features.

Model Output

The model predicts whether the input image belongs to:

Nike
Adidas

Prediction is based on probability values generated by the softmax output layer.
